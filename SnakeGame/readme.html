TUTORIAL

ONE PLAYER MODE
1. Click on P1 start

2. To move a snake,
UP = I
DOWN = K
LEFT = J
RIGHT = L

3. Blue Square : Food
Green Square : Cure
Red Square : Poison

4. If you are poisoned,
you must find a true green cure
moving reverse
(You will see what I mean)
while poisoned, you move backwards and you are green

5. Top 30 scores
If your score is above rank 30 score,
it shows a input alert that asks you to type


 TWO PLAYER MODE
1. Click on P1 vs P2 start

2. To move snakes,
UP = I
DOWN = K
LEFT = J
RIGHT = L

UP = W
DOWN = S
LEFT = A
RIGHT = D
3. Blue Squre : Food
4. Snakes lose healths if collision happens
So try not to collide.

5. End
When a player lose all health, the game ends.
